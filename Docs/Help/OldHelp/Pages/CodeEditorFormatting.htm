Code Editor Formatting



Case convert Ctrl+F12

Case converts the whole file or current selection. The rules for case converting is set by Options / Code Editor / Auto Case Convert.

Case converting is dependant on currently loaded api files.

 

Indent Shift+Ctrl+F12

Indents and trims trailing spaces for the whole file. Formatting rules are set in FbEdit.ini section [AutoFormat]

Indenting is dependant on collapseable blocks so section [Block] is also used.
