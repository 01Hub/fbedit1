Adding files to a project



Adding new file.

 

Select New File from the Project menu. Add the file as a module only if you want to bild that file separatly. Type the filename in the Add New File dialog and push Save. You will be warned if the file exists.

 

Adding existing file.

 

Select Existing File from the Project menu. Add the file as a module only if you want to bild that file separatly. Select the file in the Add Existing File dialog and push Open.
