Code Editor Find text



Find text dialog.

 

 

Find what                     The text to search for

Match Case                  If checked the search is case sensitive.

Whole Word                If checked only whole words will be found.

Skip Comment Line      If checked comments and strings will be skipped.

Log Operation              If checked operation will be logged to output window.

 

Direction

All                                Automatically starts from beginning when end is reached.

Up                               Will search upwards only.

Down                           Will search downwards only.

 

Search

Current Selection           If selected current selection will be searced.

Current Procedure         If selected current procedure will be searced.

Current Module             If selected current file will be searced.

All Open Files               If selected all open files will be searced.

All Project Files            If selected all project files will be searced.

 

Buttons

Find                             Will search for next occurance.

Replace                        Will enter replace mode.

Find all                         Only active if Log operation is checked.

Cancel                          Will close the find dialog.
