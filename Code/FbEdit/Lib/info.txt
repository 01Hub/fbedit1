=========

http://www.microsoft.com/en-us/download/details.aspx?id=21138


Extract "htmlhelp.lib"  (34904 bytes, MD5 = 190f479a7400722bf03ea392f89c78f3)
and rename to "libHtmlHelp.a"


Tre (regex)
============

Build the included (slightly modified) Tre 0.8.0 source using Makefile.

Mods from original Tre:
 - removed autotools, added custom makefile
 - removed "tre_" prefix for some functions (see tre-config.h) to be compatible
   with old lib and freebasic headers
