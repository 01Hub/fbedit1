Style manager



To show this dialog click on the button in property xStyle or xExStyle.
Here you can change the control style or extended style.
Each control class has a different style set.
The WS and WS_EX styles are common for all controls.

NOTE!
With the style manager it is possible to turn a button into a checkbox or any other control with the same class.
Don't do that as it is only a source for human confusion.
