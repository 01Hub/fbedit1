Accelerator Table Editor



 

● Add button.

Adds a new line.

● Delete button.

Deletes the selected line.

● Name.

The accelerators name.

● ID.

The accelerators ID.
NOTE!
If the name is defined elswhere the ID can be 0.

● Key.

The accelerator key.
NOTE!
If .Ascii is selected then the Ascii cell contains the ascii character.

● Ascii.

If the Key cell is .Ascii then this cell contains the ascii character.

● Ctrl.

Checked if control key should be down.

● Shift.

Checked if shift key should be down.

● Alt.

Checked if alt key should be down.



Accelerator table properties

● (ID).

The accelerator tables ID.

● (Name).

The accelerator tables name.

● Language.

The accelerator tables language.
