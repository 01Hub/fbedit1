Edit Menu



 

● Undo.

Undo last change.

● Redo.

Redo last change.

● Cut.

Copies the control(s) to the clipboard and deletes the control(s).

● Copy.

Copies the control(s) to the clipboard.

● Paste.

Paste the control(s) on the clipboard to the current dialog.

● Delete.

Deletes selected control(s).

● Find.

Text mode only. Shows the find dialog.

● Replace.

Text mode only. Shows the replace dialog.
