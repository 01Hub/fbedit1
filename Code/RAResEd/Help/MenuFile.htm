File Menu



 

● New Project.

Create a new project.

● Open Project.

Open existing progect.

● Open Project As Text.

Open existing project in text edit mode.

● Close Project.

Close current open project.

● Save Project.

Saves the current open project.

● Save Project As.

Saves the current open project with a new filename.
The Unicode checkbox lets you save a file in unicode format.
It can be used to convert a file to or from unicode

● Exit.

Exits ResEd.

● Mru Projects.

Lists the most recent used projects.
