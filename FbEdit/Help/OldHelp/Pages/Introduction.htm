Introduction to FbEdit



FbEdit is a RAD IDE written by KetilO. It includes all of the tools you need to begin writing programs in FreeBASIC.

 

Features.

 

 ● Syntax hiliting.
 ● Themes.
 ● Visual dialog editor.
 ● Custom controls.
 ● Code complete

 ● Code tooltip.
 ● Smart indent.

 ● Auto block complete.
 ● Auto case convert.
 ● Addins.
 ● Projects.
 ● Templates.
 ● Flexible build options.
 ● Uses ini file, nothing in registry.
