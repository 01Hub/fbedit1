Code Editor Accelerator Keys



Ctrl+A                         Select all text.

Ctrl+B                          Toggle block mode.

Shift+Ctrl+B                 Shows the block insert dialog.

Ctrl+C                         Copy selected text to clipboard.

Ctrl+E                          Expand / Collapse code block.

Ctrl+F                          Shows the Find dialog.

Ctrl+G                         Shows the Goto dialog.

Ctrl+L                          Converts selected text to lowercase.

Ctrl+R                          Shows the Replace dialog.

Ctrl+U                         Converts selected text to uppercase.

Ctrl+V                         Paste text from clipboard.

Ctrl+W                        Toggle full screen editing.

Ctrl+X                         Cut text to clipboard.

Ctrl+Y                         Redo.

Ctrl+Z                          Undo.

F2                                Find declare.

Ctrl+F2                        Return from Find Declare.

F3                                Find next.

Ctrl+F3                        Find previous.

F9                                Comment selected text.

Ctrl+F9                        Uncomment selected text.

 

Ctrl+Left                      Move to start of word.

Shift+Ctrl+Left             Select to start of word.

Ctrl+Right                    Move to end of word.

Shift+Ctrl+Right           Select to end of word.

Ctrl+Up                       Scroll window.

Ctrl+Down                   Scroll window.

Shift+Up                      Select line.

Shift+Down                  Select line.

 

Insert                            Toggles insert / overwrite mode.

Shift+Insert                   Paste text from clipboard.

Alt+Ctrl+Insert             Show / Hide line numbers.

Ctrl+Delete                  Delete to end of word.

Shift+Delete                 Cut text to clipboard.

Shift+Ctrl+Delete         Delete to end of line.

Home                           Goto start of line.

Ctrl+Home                   Goto start of text.

Shift+Home                  Select to start of line.

Shift+Ctrl+Home          Select to start of text.

Alt+Ctrl+Home            Jump to split.

End                              Goto end of line.

Ctrl+End                      Goto end of Text.

Shift+End                     Select to end of line.

Shift+Ctrl+End             Select to end of text.

Alt+Ctrl+End               Split window.

PageUp                        Goto previous page.

Ctrl+PageUp                Scroll to previous page.

Shift+PageUp               Selectl to previous page.

Alt+Ctrl+PageUp         Collapse blocks.

PageDown                   Goto next page.

Ctrl+PageDown           Scroll to next page.

Shift+PageDown          Selrct to next page.

Alt+Ctrl+PageDown     Expand blocks.
