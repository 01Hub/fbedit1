Creating New Project



To create a project select New Project from the File menu and the following dialog will show.

 

 

Project Name will be used to create sub folder and main files. Enter a short Project Description. Optionally change the Project Path.

Select what folders and files you want to be created. With Project Type you choose what build method to use.

 

Template.

 

 

If you want to use a template then click the Template tab and select one from the list. The lower pane will show a description of the selected template. Click OK to create the project.
