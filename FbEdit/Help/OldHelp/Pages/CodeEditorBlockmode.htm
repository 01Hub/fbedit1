Code Editor Block mode



Block mode is useful whenever you need vertical editing. When in block mode the caret will be 3 pixels wide and have a 1 pixel wide selection back color in the middle. You can do a block selection that is 0 characters wide. A 0 character wide selection will be shown as 1 pixel wide selection back color.

Block insert dialog.

 

With this dialog you can insert / replace text at every line in a vertical selection.
