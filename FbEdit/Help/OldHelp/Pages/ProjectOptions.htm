Project Options



Select Project Options from Project menu and the following dialog will show.

 

 

With project options you can change most of the project settings. You can change Project Description, Project Type and Build Command. You can also choose to Add main files to Build Command or not and change Modul Build Command and build method. You can specify an Output filename, Run Command line parameter and Project File Grouping.

 

When pushing the Build Command button the following dialog will show.

 

 

Here you can change the build command and add optional builds like a debug build.

 

If you push the Api files button the following dialog will show.

 

 

Here you choose what api files to load with your project. Api files are used by syntax hiliter, code complete and code tooltip.
