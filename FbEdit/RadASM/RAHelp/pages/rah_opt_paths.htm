Setting paths

All the paths in RadASM are set to default values when it is installed. You may want to change these paths depending on the particular setup of your system. To do this select Set Paths from the Options menu. The paths that RadASM needs are all listed, you can use the $_ variables in path names you enter in this dialog. All paths used by RadASM must be correct otherwise unforeseen errors may occur. When you first install RadASM you should verify that your paths are correct. Since these paths are stored in the Assembler.ini file they are assembler specific so you can switch between assemblers without resetting the paths.
