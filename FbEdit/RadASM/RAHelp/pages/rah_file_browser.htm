Using the File Browser

The File Browser is available from the Project window. The toolbar at the top of this window contains the following buttons from left to right:

Project Browser : Changes the window view to the project browser
Project Groups : Toggles showing project files in group folders
File Browser : Changes the window view to the file browser
File Filter : Toggles file filtering on or off
Up one level : Moves the view up one level in the directory tree
Previous/Next Folder : Cycles through the default folders set in File browser options

Pressing the File Browser Button in the Project Window will start browse mode. The folder that will be displayed is the first path on the list of folders set in the File Browser options. Use the green arrows to page through the list of folders that you specified in the options. The file type filter can be turned on and off using the filter button.
