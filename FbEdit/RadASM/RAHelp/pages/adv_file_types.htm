Adding to the default file types

All RadASM command lines used to build a project use $ variables to dynamically specify  a file name. This list is kept in your RAP file and can be modified by using the Main Project Files selection from the Project menu. There are 20 available file types though 0 through 11 are generally used for the RadASM defaults.

As an example where this might be useful is in creating an OCX file. You would add Project.ocx to the list in for example position 12 then change the link command to the following:

12,O,$B\LINK.EXE /SUBSYSTEM:WINDOWS /DLL /OUT:$12 /DEF:$6 /LIBPATH:"$L",3,4
