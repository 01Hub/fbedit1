Color Reference by minor28

Color Reference is an alternative to the insert color AddIn. It features an intereface fully integrated with the ChooseColor dialog and has some added functionality. The main feature of this color picker is the capture color button. You can capture any color on the screen and convert it to a 3 part RGB color, hex or decimal representation. The custom colors are saved to the project and main ini file, the top row of colors are global and available to all projects, the lower row is specific to the current project.

To install the AddIn add the following toto [AddIns] section of RadASM.ini and copy the colorref.dll file to the AddIns folder.

x=colorref.dll,y

Replace x with next unused number.
Replace y with number:
0 = deactivated
1 = active
