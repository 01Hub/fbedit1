Add the Following line to RadASM.INI

In the [AddIns] section

XX=ButtonChanger.dll,Y

Where XX is the next available AddIn number.

Copy ButtonChanger.dll to the AddIns folder

Values for Y
0 = keep default icons
1 = extract icons from executables
2 = use ICO files
BCIconA.ICO
BCIconB.ICO
These ICO files are placed in the AddIns folder
