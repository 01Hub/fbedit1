Adding file to a project

There are times when you get to a point in a project where you must add a file. These functions are handled in the Project menu. The Add New menu item in the Project menu gives you several types of files to add. In cases where you add a resource such as a menu the necessary #include is made to the main RC file. You can also access the Project menu by right clicking in the Project box. When added the file will be placed in the Added Files group, to change the group that it is in or create new groups use the group editor.

TIP: When opening a project it is possible to have files loaded automatically. Just right click on the file in the project box and select Auto Load, when the project is opened the file will be loaded as well.
