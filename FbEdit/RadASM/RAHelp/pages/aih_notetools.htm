NoteTools By Donkey

This AddIn will allow you to set a 127 byte comment for anything in the properties window. Use the context menu to set the comment and it will appear as a tooltip when you hover over the item at the cursor. Install it like any other AddIn.

XX=NoteTools.dll,YY

Where:

XX is the next available AddIn number
YY is an option number that is passed to the AddIn. Set this to 0 to disable the AddIn

Options:

1 = NoteTools enabled
2 = Use background color for NoteTools tip
4 = Use title for NoteTools tip
8 = Use background color for RadASM tip
16 = Use title for RadASM tip

Add the values to get combinations,15 is a nice combination. You can still set the ToolTip options for RadASM if NoteTools is disabled.
