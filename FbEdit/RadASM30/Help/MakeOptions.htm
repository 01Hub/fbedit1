Options



 Make Options.

The listbox will show all make types.

Adding a new make type:
Type a short description of your make type in the textbox below the Add New button.
Push the Add New button and it will appear in the listbox.
Fill in the commandline switches and Output in Compile RC, Assemble, Link and Library.
To get help on commandline switches, push the [?] in the dialogs caption, then click in the textbox.
The Output can contain relative or absolute paths, but it is recomended that these are added using project options.

Acronymes used in Commandline switches:
$C Main codefile.
$D The projects def file used to build dll's.
$F Main file without extention.
$M Modules. When linking, and your project has modules, a Mod.txt command file is created and added to the command line.
$O Resulting output file.
$R Main resource script file.

Acronymes used in Output:
$C Main codefile without extention.
$R Main resource script file without extention.

External debugger:
Here you can add an external debugger. If you added the path to the debugger to the environment then you only need the filename.
