Options



 Code Editor Options.

● Color themes.

    Here you can select a color theme or create a new color theme.

● Colors.

    Here you can select colors for the editor and other tools.

● Syntax.

    Here you can add / remove words to be highlighted.
    You can also choose bold and / or italic font and also set if the group should be used for RC files only.
    Code files: Here you specify file extentions for files that should be treated as code files. Note the use of dots.

● Misc.

    Tab size: Number of spaces.
    Expand tabs: Tabs will be expanded to spaces while typing.
    Show linenumbers: The linenubers bar will be shown when you open a file.
    Auto Brace: The oposit bracket wil automatically be added.
    Hilite active line: The current line will be highlighted.
    Auto indent: The indent will be copied from the line abowe.
    Hilite comments: Comments will be highlighted.
    Restore session: The last project or files will be loaded on startup.
    Show Codetip: A tooltip will be shown when the mouse hoovers a known word.
    Multiline tabtool: If checked the tabtoll wil be multiline.

● Fonts.

    Lets yo choose fonts for Code, Text and Hex editor and tool windows.
