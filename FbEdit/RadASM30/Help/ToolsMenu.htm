ToolsMenu

Options



 Tools Menu.

● Menu item:

    Here you specify the text shown on the tools menu.
    Note that a dash creates a separator.

● Command:

    The tool to run.

● Acronyms:

    $A=Application path (path to RadASM.exe).
    $=Current open file without extention.
    $C=Main code file.
    $R=Main resource file.
    $P=Project file.
