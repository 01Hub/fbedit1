New Project



 

● Programming Language:

Select the programming language from the combobox.

● Project Name:

Type in a name for your project. The name will be used as a filename.

● Project Description:

Enter a short description of your project.

● Project Path:

Browse for the project path.

● Folder Creation:

If Project Subfolder is checked a Project Name folder will be created.

● Files tab:

Check the files you want to be created.

● Build tab:

Select the build type(s) for your project.

● Template tab:

Select a template to use. Note that if a template is used then selections on the Files and Build tabs wil be overridden.
