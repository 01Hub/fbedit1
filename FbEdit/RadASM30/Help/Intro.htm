Introduction to RadASM



 Code Editor.

 Resource Editor.

 Features.

 

RadASM supports the following programming languages:

 

Assembly:

● Masm.

● GoAsm.

● Fasm.

● Tasm.

● JWasm.

● sol_asm.

● asemw (8051 assembler).


High level:

● VC6 C/C++.

● BCC 5.5 C/C++.

● MinGW C/C++.

● Free Pascal.

● FreeBASIC.


The RadASM code editor:

● Syntax highlighting.

● Code complete and code tooltips.

● Collapseable code blocks, saved to project file.

● Bookmarks, saved to project file.

● Breakpoints, saved to project file (Masm).

● Auto block complete.

● Source code debugger (Masm).

● Project browser with project groups and sub groups.


The RadASM resource editor handles the following resource types:

● Dialogs with more than 30 windows controls + custom controls.

● Menus.

● String tables.

● Accelerator tables.

● Version info.

● Language setting.

● Resources like bitmaps, icons, manifest and several others.

● Unlimitted Undo / Redo on dialog editor.

● Several pre defined name export formats.

● User defined name export format.

● Can read and write unicode files.
