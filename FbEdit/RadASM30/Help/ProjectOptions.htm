Project Options



 

● Make Options:

 ● Listbox:

 Select a make type from the list to change its settings.

 ● Add Existing:

 Select a make type from the combobox and push the [Add Existing] button.

 ● Add New:

 Type in a name for the make option and push the [Add New] button.
 Fill in the command lines and resulting output

 ● Delete:

 Select a make type from the list and push the [Delete] button.
 Note that you cannot delete the last remaining make type.

 ● Delete Minor Files After successful Build:

 If checked then minor files will be deleted if the build did not had any errors.

 ● Increment Build:

 If yor project has a resource file and the resource file has a versioninfo then the build will be incremented.
 Note that the resource file must be open.

● Run Options:

 ● Run Commandline Parameter:

 Type in the parameters.

 ● Start Cmd.exe:

 If checked and the textbox contains 'Cmd /k' then a command window will be opend.
 This is useful if your commandline has redirections of input or output.

● Api Files:

Type in a comma separeted list of api files you want to be loaded with your project.
