Format Menu



 

● Lock Controls.

Locks / Unlocks controls on the dialog.

● Send to Back.

Sends current selected control to the back.

● Bring to Front.

Brings the current selected control to front.

● Show Grid.

Shows / Hides the dialog grid.

● Snap to Grid.

Snaps controls to the grid.

● Align Controls.

Left, Center, Right. Horizontal alignment.
Top, Middle, Bottom. Vertical alignment.

● Same Size.

Width. The width of all selected controls will be the same.
Height. The height of all selected controls will be the same.
Both. The width and height of all selected controls will be the same.

● Center in Dialog.

Horizontal. All selected controls will be centered horizontally.
Vertical. All selected controls will be centered vertically.

● Set tabindex.

Lets you set the tabindex for the controls om the dialog.
Use Ctrl+Left click to start at an offset.

● Auto renumber ID's.

Renumbers ID's for selected controls.
