Options



 Export options.

● Name Export Format.

    Assembly equ's: IDC_BUTTONADD equ 1001
    C defines: #define IDC_BUTTONADD 1001
    hla: IDC_BUTTONADD := 1001;
    PureBasic: #IDC_BUTTONADD = 1001
    PowerBasic: %IDC_BUTTONADD = 1001
    User Defined: %name and %id will be replaced by the actual values.

● Name Export Output.

    File: The names are written to file.
    Clip Board: The names are pasted to clipboard.
    Output: The names are sendt to output window.

● Default Export Filename.

    Name of file if File is selected.

● Auto export when saved.

    If checked the export will be done when the project is saved.
